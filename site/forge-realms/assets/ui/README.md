# UI Splash Assets

These images are project-bound GPT Image 2 / ChatGPT Images 2.0 style generations used by the game shell.

## Files

- `start-page-v1.png`: opening menu background.
- `victory-screen-v1.png`: victory and defeat result background.
- `start-page-v1.webp`: runtime-optimized derivative generated from the PNG.
- `victory-screen-v1.webp`: runtime-optimized derivative generated from the PNG.

Keep the PNG files as the canonical high-quality sources. Regenerate WebP runtime derivatives with:

```bash
npm run images:runtime
```

## Start Page Prompt

Create an awesome title/start screen background for a late-1990s-inspired Stone Age RTS game called Forge Realms. Show a top-down/isometric ancient settlement at dawn with a thatched town center, villagers gathering wood and berries, clubmen and spearmen guarding a dirt path, coastline and shallow blue sea in the distance, and warm torchlight. Leave the center-left and lower-center clean for overlaid title and buttons. Premium hand-painted pixel-art / illustrated RTS splash art, crisp historical game feel, coherent with isometric Stone Age building sprites. No text, logos, UI, watermark, modern objects, fantasy magic, photorealism, or blur-heavy stock art.

## Victory Prompt

Create a triumphant victory screen background for a late-1990s-inspired Stone Age RTS game called Forge Realms. Show a conquered enemy town center at sunset, player villagers and soldiers celebrating around banners, rebuilt thatched buildings, campfires, food baskets, wood piles, stone and gold tribute, coastline and ships in the distance. Leave a calm central area for overlaid result text and restart button. Premium hand-painted pixel-art / illustrated RTS splash art, crisp historical strategy game feel, consistent with Stone Age isometric building sprites. No text, logos, UI, watermark, gore, modern objects, fantasy magic, or photorealism.
